Detailed Description

Singleton manager for the coin economy.

Holds the current coin balance and keeps the HUD text in sync on every change. Access from any script via CoinManager.Instance.

Note
Only one CoinManager should exist in the scene at a time. No duplicate protection is implemented — if multiple instances exist, the last one to Awake wins.

void CoinManager.UpdateCoins ( int changeAmount)

Adds or subtracts coins and refreshes the HUD text immediately.

Pass a positive value to award coins, negative to spend them. Does not guard against the balance going below zero.
