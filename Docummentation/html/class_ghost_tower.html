Detailed Description

Renders a semi-transparent tower preview that changes color based on whether the tile is valid.

Green = valid placement. Red = invalid placement. Alpha is set to 50% on Awake to distinguish the ghost from a real placed tower.

void GhostTower.SetValid ( bool isValid)

Updates the ghost color to reflect placement validity.

Note: this overwrites the alpha set in Awake, making the ghost fully opaque when colored. Consider preserving alpha if a tinted semi-transparent look is desired.
