\doxysubsection{Detailed Description}
Singleton manager for the coin economy. 

Holds the current coin balance and keeps the HUD text in sync on every change. Access from any script via \doxylink{class_coin_manager_af25bc29bec09deb4115afbd9aa49530a}{Coin\+Manager.\+Instance}.

\begin{DoxyNote}{Note}
Only one \doxylink{class_coin_manager}{Coin\+Manager} should exist in the scene at a time. No duplicate protection is implemented Ś if multiple instances exist, the last one to Awake wins. 
\end{DoxyNote}
