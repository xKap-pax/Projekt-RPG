\doxysubsection{Detailed Description}
Reads mouse input each frame to show a placement ghost and place towers on valid tiles. 

Requires two Tilemaps:\+ one for valid placement tiles and one marking blocked zones. Works in conjunction with \doxylink{class_tower_selection}{Tower\+Selection} to know which prefab to place.
