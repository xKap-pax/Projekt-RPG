\doxysubsection{Detailed Description}
UI controller that tracks which tower prefab the player has selected. 

Attach to each tower selection button\textquotesingle{}s Game\+Object. Uses a static field so \doxylink{class_tower_placer}{Tower\+Placer} can read the selection without a direct reference.
